# inflearn
인프런 교육 (실전! 스프링 부트와 JPA 활용1 - 웹 애플리케이션 개발)

# 목차
 * view 환경설정
 * JPA와 DB 설정, 동작확인

### 요구 사항
 * 회원 기능
   * 회원 등록
   * 회원 조회
 * 상풍 기능
   * 상품 등록
   * 상품 수정
   * 상품 조회
 * 주문 기능
   * 상품 주문
   * 주문 내역 조회
   * 주문 취소 
 * 기타 요구사항
   * 상품은 제고 관리가 필요하다.
   * 상품의 종류는 도서, 음반, 영화가 있다.
   * 상품을 카테고리로 구분할 수 있다.
   * 상품 주문시 배송 정보를 입력할 수 있다.
